# 🎮 Busca de Jogos com API (CheapShark)

## Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina de **Linguagem de Programação** do curso de Análise e Desenvolvimento de Sistemas - Fatec Itu - Don Amaury Castanho.

A aplicação permite buscar jogos em promoção utilizando uma API externa, exibir os resultados em uma tabela, salvar dados no navegador e gerenciar uma lista de favoritos.

## Funcionalidades

* Buscar jogos por nome.
* Consumo de API externa (CheapShark).
* Exibição dos resultados em tabela.
* Armazenamento no localStorage.
* Sistema de favoritos.
* Exclusão de itens com confirmação.
* Carregamento automático dos dados salvos.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* API CheapShark

## API Utilizada

A aplicação consome dados da API pública:

[https://www.cheapshark.com/api/1.0/deals]

Exemplo de uso:

https://www.cheapshark.com/api/1.0/deals?title=batman
